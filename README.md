# CASE_sensor_network

## Software Installation

RPi config
* enable I2C

Create venv
`python -m venv venv`

Activate venv
`source venv/bin/activate`

Install SHT31-D sensor library
* https://github.com/adafruit/Adafruit_CircuitPython_SHT31D
* `pip3 install adafruit-circuitpython-sht31d`
