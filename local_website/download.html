Downloading Sensor Data

Sensor data can be retrieved in a number of ways. Each Raspberry Pi Zero device stores its own data on its SD card. The data is also copied to the Windows laptop.

  • The best way to retrieve data is via the API. This can be done programatically or via the sensor interfaces. Links to sensor interfaces are below.
  • You can also retrieve the files from the Windows laptop, assuming it is successfully archiving data.
  • You can SCP files from the devices directly. On all devices, the files are located in the /home/case/data directory.
  • You can manually copy CSV files directly from SD cards on device. This is less ideal, because it means that the sensor isn't collecting data while you are doing this.
